== Description ==

Remove unused shortcodes from your posts content.

If you have a lot of pages, posts or any post type that have unused shortcodes from previously used plugins, then you can remove theme using this simple tool.

== Installation ==

1. Upload `clean-unused-shortcodes.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to tools then "Clean unused shortcodes"
4. Choose you prefered post types to clean or use all, then hit clean.
